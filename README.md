# stage.nowxx

> Realtime staging environments.

This is the app that powers [stage.now.sh](https://stage.now.sh).

## Development

Install project dependencies:

```bash
$ yarn
```

Run the dev server

```bash
$ yarn run dev
```

Open [http://localhost:3000](http://localhost:3000)
